Line chart - Time line 15

In this step we are going to build a line chart that shows daily cases and deaths trending in US.

Steps for this report

1. Drag and drop line chart from visualizations.

2. Drag and drop

  • Date column from COVID-19 in US table to axis in fields box
  • Total cases from  measures table to values in fields box
  • Total deaths from measure table to secondary value in fields box.

3. Legend - on

  • Color - White, 20% darker
  • Font family - Arial
  • Text size - 10pt
  • Style - Markers only

4. X-axis - on

  • Color - White, 20% darker
  • Type -  Continuous 
  • Text size - 9pt
  • Font family - Arial
  • Display units - Auto
  • Title - off

5. Y-axis - on

  • Color - White, 20% darker
  • Text size - 9pt
  • Font family - Arial
  • Display units - Thousands
  • Title - off

6. Secondary Y-axis - on

  • Scale type - linear
  • Color - #f9bba9, theme color 8, 60% lighter
  • Text size - 9pt
  • Font family - Arial
  • Title - off

7. Data colors 

  • Total cases color -  White, 10% darker 
  • Total deaths color - #f9bba9, theme color 8, 60% lighter

8. Data label - on

  • Display units - Thousands
  • Text size - 9pt
  • Customize series - on
  • Select - Total deaths
  • Color -  #f9bba9, theme color 8, 60% lighter
  • Display units - Thousands
  • Text size - 9pt
  • Font family - Arial

9. Title - on 

  • Add a title text and use same format through out the dashboard ( Refer step 4)

10. Boarder - on

  • Color - Black, 40% lighter
